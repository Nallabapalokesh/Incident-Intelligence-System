# 🚨 Incident Intelligence System

An end-to-end data analytics project that simulates real-time incident monitoring using Python, FastAPI, PostgreSQL, and Power BI.

---

## 🔧 Tech Stack

- Python (Pandas)
- FastAPI
- PostgreSQL
- Power BI
- Streamlit
- Azure (Power BI API)

---

## 🚀 Features

- Real-time data simulation
- Data processing pipeline
- REST API integration
- Power BI dashboard
- Automated dataset refresh
- Interactive UI for pipeline control

---

## 🧱 Architecture

Data Generation → Processing → PostgreSQL → FastAPI → Power BI → UI

---

## 📊 Dashboard

- Incident trends
- Severity distribution
- Response time analysis
- Geo visualization
